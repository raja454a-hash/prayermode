---

---

### Cleanup Plan

**Phase 1 ‚Äî Remove dead code from active files:**
- Remove `updatePassword`, `session` from `useAuth.ts` return; remove `fetchProfile` from return
- Remove `getSilentModeStatus` from `nativeSilentMode.ts`

**Phase 2 ‚Äî Delete unused UI components:**
- Delete: `sonner.tsx`, `sidebar.tsx`, `navigation-menu.tsx`, `use-mobile.tsx`
- Delete unused shadcn components: `accordion, aspect-ratio, avatar, breadcrumb, carousel, chart, collapsible, command, context-menu, drawer, hover-card, input-otp, menubar, pagination, resizable, scroll-area, select, separator, skeleton, slider, table, tabs, toggle, toggle-group`

**Phase 3 ‚Äî Remove unused npm dependencies:**
- Uninstall: `next-themes`, `sonner`, `react-resizable-panels`, `embla-carousel-react`, `input-otp`, `cmdk`, `vaul`, `react-day-picker`, `recharts`

**Phase 4 ‚Äî Clean up minor issues:**
- Remove placeholder test file `src/test/example.test.ts`

**Impact:** Smaller bundle size, cleaner codebase, fewer unnecessary dependencies. No functional changes ‚Äî all existing features remain intact.
